# StudyFam
StudyFam is a simple python program for creating and joining study groups. Users can register with their skils to teach and learn , create groups to learn new skills , view member profiles , rate fellow learners

##Features 
1.user registration with skills to teach and learn
2.create skill based teaching groups
3.join existing groups to learn new skills
4.view profiles of all members
5.add rating to members
6.data saved using JSON files

##How to use
1. Run the program using python 3.
2. use the menu options to register , create/join groups , view profiles , add ratings
3. data is saved automatically after changes , so your information is kept for next time

##Requirements
1.python 3.x
2.no external libraries required

##how data is stored
data is stored in JSON file ('studyfam_data.json')
