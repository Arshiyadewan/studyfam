# StudyFam - Study group management system

## Overview
StudyFam is a python-based command line application designeed to help students for creating and joining study groups. Users can register with their skils to teach and learn , create groups to learn new skills , view member profiles , rate fellow learners. By focusing on simple yet effective group formation and transparent feedback , StudyFam makes academic collaboration accessible and engaging for everyone.

## Features 
1. user registration with skills to teach and learn
2. create skill based teaching groups
3. join existing groups to learn new skills
4. view all existing groups
5. view profiles of all members
6. add rating to members
7. Menu based interface for simple navigation

## Technologies/Tools used
1. Python 3
2. Standard python libraries
3. Command line interface for intraction
